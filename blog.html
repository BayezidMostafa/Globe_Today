QUESTION-1
WHAT IS THE DIFFERENCE BETWEEN VAR/LET/CONST?

var: var is global-scoped or functional-scoped. This variable can be declared and access without initialization. We can update and re-declared this variable into the scope.

let: let is a block-scoped variable. We can declare this variable without initialization. We can not re-declare this variable but we can re-assigned it.

const: const is a block-scoped variable. This variable can't be decalred without initialization. We can't update or de-declared this variable into the scope.

WHAT IS THE DIFFERENCE BETWEEN ARROW AND RAGULAR FUNCTION?

The most common difference between arrow and regular function is syntax. We can get the same result as regular function within few lines of code. Arrow functions don't have their own 'this' or 'arguments' binding like regular functions. Regular functions have arguments objects but, in arrow functions haven't.

QUESTION-3
WHAT IS THE DIFFERENCE BETWEEN MAP(), FOREACH(), FILTER(), ADN FIND()?

map(): The map() method create a new array with the result of calling a function for every array element. The map() will not change the original array.

forEach(): The forEach() method is used to loop through the contents of an array and show each element as output. The forEach() method will not execute for empty elements

filter(): The filter() method creates a new array filled with elements that pass a specific test provided by a function. The filter() method does not execute the function for empty elements.

find(): The find() method returns the value of the first array element that pass the provided test provided by a function. If no values pass the test, undefined is returned.

QUESTION-4
WHY WE SHOULD USE TEMPLATE STRING?

By using template literals(``), We can use both single quote('') and double quote to define a string. We can declare multiline string in it. Template literals allow us to use variables in string. It helps us to create clean JavaScript code when we have to requesting an API.
